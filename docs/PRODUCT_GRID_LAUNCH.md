# Product grid launch (archive outputs)

The FeedFoundry output set (transcript, chapters, clip candidates, show notes, manifest, export bundle) forms the **product grid** for annual hosted archive positioning.

**Launch sequencing (conceptual):**

1. **Core job spine** — upload → job → outputs JSON in R2 (current MVP).
2. **Hosted manifest** — public JSON with stable URLs (no secrets in manifest).
3. **Grid UI (Base44 or web)** — consumes OpenAPI; this monorepo stays engine-first.
4. **Hyperframes / vertical video teasers** — optional `ff_*_hf_v1` templates from Stevenson contracts for marketing clips derived from **approved** outputs only.

Cross-reference vendored template IDs: `vendor/stevenson-contracts/product_template_ids.v1.json`.
